# PHP Mail - PHPMailer

## Amaç

Bu örnek, PHPMailer kütüphanesi kullanılarak SMTP üzerinden güvenli e-posta gönderimini göstermektedir.

## Kullanılan Teknolojiler

- PHP
- PHPMailer
- SMTP
- Gmail SMTP
- JSON

## İşleyiş

1. Composer Autoload yüklenir.
2. PHPMailer nesnesi oluşturulur.
3. SMTP ayarları yapılır.
4. Gönderen ve alıcı belirlenir.
5. Konu ve mesaj hazırlanır.
6. Mail gönderilir.
7. Sonuç JSON olarak döndürülür.

## Kullanılan Sınıflar

- PHPMailer
- Exception

## Güvenlik

✅ SMTP Authentication

✅ TLS

✅ Exception Handling

✅ JSON Response
